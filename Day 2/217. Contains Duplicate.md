**Approach 1: Set**
```
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        return len(nums) != len(set(nums))
```

**Approach 2: Dictionary**
```
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        count = dict.fromkeys(nums, 0)
        for n in nums:
            count[n] += 1
            if count[n] > 1:
                return True
        return False
```

**Approach 3: Sort**
```
class Solution:
    def containsDuplicate(self, nums: List[int]) -> bool:
        nums.sort()
        for i in range(len(nums)-1):
            if nums[i] == nums[i+1]:
                return True
        return False
```
